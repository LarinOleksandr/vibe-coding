## Skills

- `$create-plan` - Create a concrete, executable plan for large or risky changes.
- `$db-migration` - Evolve Supabase schema/data safely with migrations and RLS intact.
- `$new-agent` - Add or modify an orchestrator agent/workflow with explicit state and tests.
- `$add-api-endpoint` - Add or modify API/Edge endpoints with contracts and validation.
- `$export-doc` - Export validated agent outputs into user-facing documents.
- `$architecture-review` - Review cross-boundary architecture changes for compliance and risks.
- `$context-maintenance` - Maintain project context, knowledge, and methodology; keep routes consistent and remove duplication.
- `$close-thread` - Close a thread with validation status and documentation updates.
- `$refactor` - Perform structural changes without altering external behavior.
- `$bug-fix` - Diagnose and fix well-scoped defects or regressions.
- `$feature-development` - Turn feature requests into confirmed, actionable briefs.
- `$dev-debug-workflow` - Run the dev stack, inspect via MCP DevTools, and resolve browser errors.
