# Conversations

This folder stores conversation summary artifacts created via `$conversation-save`.

## What goes here

- `YYYY-MM-DD__<thread-code>__<slug>.summary.md` - committed, short, curated memory.
- `YYYY-MM-DD__<thread-code>__<slug>.transcript.md` - optional full transcript (ignored by default via `.gitignore`).

## Keep it safe

- Follow `KB_REPOSITORY_RULES` (especially secrets-outside-git).
- Prefer adding durable decisions to `DOC_PROJECT_INSIGHTS` and link the summary from there.
