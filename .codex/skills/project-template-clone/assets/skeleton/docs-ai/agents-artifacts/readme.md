# docs-ai/agents-artifacts

This folder stores agent-generated artifacts for this project.

## Structure

- `plans/` contains planning files created by `$plan-creation`
- `design-reviews/` contains design review artifacts from `$design-review`
- `prds/` contains PRDs created by `$brainstorming`
- `research-briefs/` contains research briefs created by `$feature-research`
- `screenshots/` stores UI validation screenshots (README only)
- `traces/` stores Playwright traces (README only)
- `videos/` stores Playwright videos (README only)
- `playwright-report/` stores generated Playwright reports (README only)
- `conversations/` contains conversation summaries created by `$conversation-save`

## Rules

- One artifact per task.
- Do not overwrite old artifacts.
